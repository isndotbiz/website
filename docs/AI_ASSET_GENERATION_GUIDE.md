## Implementation Guide

### Recommended Services for Generation

1. **Flux 2 Pro via fal.ai** (Best Quality)
   - API: https://fal.ai/flux
   - Best for: Hero backgrounds, portfolio mockups
   - Cost: ~$0.05-0.10 per image

2. **Flux 2 via Black Forest Labs API**
   - API: https://bfl.ai/
   - Best for: All asset types
   - Official source, high quality

3. **Alternative: DALL-E 3 via OpenAI**
   - API: https://platform.openai.com/
   - Good for: Icons, simpler designs
   - Cost: $0.04-0.08 per image

4. **Alternative: Midjourney via Discord**
   - Manual generation via Discord bot
   - Best for: High-end artistic assets
   - Requires subscription

### Generation Tips

1. **Color Consistency**: Always specify exact hex colors in prompts
2. **Resolution**: Request specific resolutions in the prompt
3. **Batch Generation**: Generate 2-3 variations of each asset
4. **Quality Keywords**: Include "professional", "high quality", "ultra detailed"
5. **Negative Prompts**: Always use to avoid unwanted elements
6. **Iterations**: Be prepared to refine prompts based on results

### Post-Processing

After generation:
1. **Optimize for Web**: Use ImageOptim or similar tools
2. **Create WebP Versions**: For better performance
3. **Generate Responsive Sizes**: Create @2x and @3x versions where needed
4. **Test Colors**: Verify brand colors are accurate
5. **Accessibility**: Ensure sufficient contrast for overlaid text
