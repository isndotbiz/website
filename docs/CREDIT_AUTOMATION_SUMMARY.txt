================================================================================
1PASSWORD CLI INTEGRATION
================================================================================

HOW IT WORKS:
  1. Script validates 1Password CLI is installed and authenticated
  2. Retrieves credentials: op item get "Equifax Business" --vault Research
  3. Extracts username/email and password fields
  4. Uses credentials for browser automation
  5. Credentials NEVER stored in code or config files

SETUP:
  # Sign in
  eval $(op signin)
  
  # Verify
  op account list
  
  # Create items
  op item create --category=Login --title="Equifax Business" \
    --vault="Research" username="email" password="pass"

TROUBLESHOOTING
================================================================================

"Not signed in to 1Password"
  → eval $(op signin)

"Item not found"
  → op item list --vault Research | grep -i equifax
  → ./setup_1password_items.sh

"Browser doesn't open"
  → ./venv/bin/playwright --version
  → source venv/bin/activate && playwright install chromium

"2FA timeout"
  → Complete 2FA within 5 minutes
  → Have phone ready before starting

"Selectors not working"
  → Update config.json with new selectors
  → Run without --headless to debug
