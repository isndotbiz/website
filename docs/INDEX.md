## FAQ

**Q: Can I use individual components or must I use all 7?**
A: Use any combination. All components are modular and independent.

**Q: Do I need to generate videos with AI?**
A: No, you can use any video files. AI prompts are optional suggestions.

**Q: Will this work with my existing CSS?**
A: Yes, components use namespaced classes to avoid conflicts.

**Q: What's the browser support?**
A: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+, iOS 14+, Android 90+

**Q: Is this accessible?**
A: Yes, WCAG 2.2 Level AA compliant with proper implementation.

**Q: Can I customize the colors?**
A: Yes, all colors use CSS custom properties. See customization guide.

**Q: How big are the files?**
A: HTML ~50KB, CSS ~30KB, JS ~40KB. Videos depend on optimization.

**Q: Do I need a CDN?**
A: Libraries use CDN. Your videos can be self-hosted or use CDN.
