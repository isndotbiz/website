# GPT-IMAGE-1.5 PROMPTS FOR ISN.BIZ (ALL NEW ASSETS)

This prompt set follows the OpenAI gpt-image-1.5 prompting guide patterns:
- Structured prompts: background/scene -> subject -> key details -> constraints + intended use.
- Photorealism via camera/lens/lighting terms and natural texture cues.
- Explicit preserve/change rules for edits.
- Cost control: quality="low" for ALL generations/edits.

GLOBAL CONSTRAINTS (apply to all non-logo images)
- No watermarks, no text, no logos, no trademarks, no UI brand names.
- Use brand palette accents only: #1E9FF2 blue, #5FDFDF cyan, #3F4447 charcoal.
- Clean, premium, enterprise aesthetic. Avoid clutter.
- Leave negative space where hero text will overlay.

OUTPUT SIZES (suggested)
- Hero/backgrounds: 1920x1080 (16:9)
- Portfolio and service visuals: 1600x900 (16:9)
- Founder headshots: 1200x1200 (1:1)
- Logos: 2000x2000 (square) and 2600x800 (horizontal)
