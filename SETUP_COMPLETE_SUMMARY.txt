TROUBLESHOOTING
================================================================================

If you see: "FAL_KEY environment variable not set"
  Run: eval $(op signin)
       export FAL_KEY=$(op read "op://Research/FAL API Key/credential")

If you see: "Virtual environment not found"
  Run: python3 -m venv venv_fal
       source venv_fal/bin/activate
       pip install fal-client requests

If you see: "No module named 'fal_client'"
  Run: source venv_fal/bin/activate
       pip install fal-client requests

See PHOTO_ENHANCEMENT_README.md for more troubleshooting.
