GEDCOM Processing & Data Integrity Platform

Production-grade genealogy data cleaning with zero data loss guarantees

Overview

Genealogy data is high-value but messy. This system delivers an automated, integrity-first pipeline that transforms raw GEDCOM files into standardized, import-ready assets while preserving every relationship.

Key Capabilities

  • GEDCOM scanning, issue detection, prioritized reporting
  • Date normalization, name and place standardization
  • Relationship-preserving deduplication and safe merges
  • End-to-end processing with backups, verification, rollback

Differentiation

  • Integrity-first design with explicit verification
  • Auditable processing with documented change trails
  • Packaged CLI and library for integration

Technology Stack

Python 3.11, ged4py, click, rapidfuzz, python-dateutil
